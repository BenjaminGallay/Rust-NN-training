# What is this project about ?
This is a personal mini-project for a research project about neural-networks verification using formal methods. It uses the Burn library to train feed-forward ReLU-activated neural networks on a linear specification and a randomy generated dataset. This si done to test formal verification methods on various sizes and amount of training of neural networks.

# Exporting the model
After the training, the model is exported to /models/prettymodel.json
